# Attribution

## Paper and Dataset

This project is based on:

Eduard Gabriel Poesina, Cornelia Caragea, and Radu Tudor Ionescu. 2024. *A Novel Cartography-Based Curriculum Learning Method Applied on RoNLI: The First Romanian Natural Language Inference Corpus*. Proceedings of ACL 2024, pages 236-253.

- ACL Anthology page: https://aclanthology.org/2024.acl-long.15/
- Official repository: https://github.com/Eduard6421/RONLI

The official repository states that it contains the RoNLI dataset and baselines introduced in the ACL 2024 paper and releases the code/data under a Creative Commons Attribution Non Commercial Share Alike 4.0 license. This course project uses the data for educational reproduction and benchmarking.

## External Libraries

- pandas and numpy: data manipulation
- scikit-learn: TF-IDF, dummy baseline, logistic regression, SVM, metrics
- PyTorch: custom MLP training
- Hugging Face Transformers: optional pretrained multilingual transformer fine-tuning
- matplotlib: figures and training curves
- joblib: saving sklearn models
